## Installation
To get a local copy up and running, follow these simple steps:

1. Clone the repository:
    ```sh
    git clone https://github.com/mossimka/WebDevMusic.git
    ```
2. Navigate to the project directory:
    ```sh
    cd WebDevMusic
    ```
